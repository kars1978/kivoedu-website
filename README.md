# KivoEdu Website

Marketing and landing site for the KivoEdu platform. Built with Next.js 16 + Tailwind CSS 4.

---

## What This Is

Public-facing website for Kivo — the AI-powered school operating system. Communicates the product vision, targets school administrators and teachers, and drives sign-ups for early access.

---

## Stack

| Tool          | Version | Purpose              |
| ------------- | ------- | -------------------- |
| Next.js       | 16      | App Router, SSR/SSG  |
| React         | 19      | UI                   |
| Tailwind CSS  | 4       | Styling              |
| TypeScript    | 5       | Types                |

---

## Dev Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

---

## Scripts

| Command           | Action                           |
| ----------------- | -------------------------------- |
| `npm run dev`     | Local dev server with hot reload |
| `npm run build`   | Production build                 |
| `npm run start`   | Serve production build locally   |
| `npm run lint`    | ESLint                           |
