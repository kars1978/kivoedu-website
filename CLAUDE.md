# CLAUDE.md -- kivoedu-website

Marketing and landing site for KivoEdu. Next.js 16 + Tailwind CSS 4 + TypeScript.

---

## What This Is

Public-facing website for Kivo — the AI-powered school operating system.
Targets school admins, teachers, and parents. Drives sign-ups for early access.

---

## Stack

- **Next.js 16** — App Router
- **React 19**
- **Tailwind CSS 4**
- **TypeScript 5**

---

## Dev

```bash
npm run dev    # http://localhost:3000
npm run build
npm run lint
```
